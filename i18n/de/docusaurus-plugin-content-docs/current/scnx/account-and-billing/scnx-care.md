# SCNX Care

Bei ScootKit glauben wir, dass es unsere Verantwortung ist, Dinge wieder in Ordnung zu bringen, wenn etwas schiefläuft. **SCNX Care** ist unser Kulanzprogramm für Kunden, das dazu dient, Serviceunterbrechungen anzuerkennen, Probleme transparent zu lösen und sicherzustellen, dass du dich als Mitglied unserer Community wertgeschätzt fühlst.

## Was ist SCNX Care?

Wenn du ein Serviceproblem auf SCNX erlebst – sei es unerwartete Ausfallzeiten, ein Software-Bug, eine Abrechnungsunstimmigkeit oder eine verzögerte Support-Antwort – kann unser Team dir einen **Care-Gutschein** ausstellen. Dieser Gutschein ermöglicht es dir, eine kostenlose Entschädigung als unsere Entschuldigung für die Unannehmlichkeiten auszuwählen.

## So funktioniert es

1. **Wir identifizieren das Problem:** Unser Support-Team prüft deinen Fall und bestimmt die geeignete Lösung.
2. **Du erhältst einen Care-Gutschein:** Du bekommst eine personalisierte E-Mail von `care@scnx.xyz`, die erklärt, was passiert ist, was wir getan haben, um es zu beheben, und einen Link zum Einlösen deines Gutscheins.
3. **Du siehst außerdem ein Banner** auf deinem SCNX-Dashboard, das dich darüber informiert, dass ein Gutschein auf dich wartet.
4. **Entschädigung auswählen:** Besuche die Einlöseseite und wähle die Option aus, die am besten zu dir passt.
5. **Sofort angewendet:** Die gewählte Entschädigung wird sofort auf dein Konto angewendet.

## Welche Entschädigungen sind verfügbar?

:::info
Die spezifischen Optionen, die dir zur Verfügung stehen, hängen von deinem aktuellen Plan und der Art des Problems ab. Du wählst **eine** Entschädigung pro Gutschein.
:::

Je nach Schwere des Problems kannst du eine der folgenden Optionen angeboten bekommen:

| Entschädigung                     | Beschreibung                                                            | Verfügbarkeit                                  |
| --------------------------------- | ----------------------------------------------------------------------- | ---------------------------------------------- |
| **Plan-Verlängerung**             | Zusätzliche Tage werden deinem aktuellen Plan ohne Aufpreis hinzugefügt | Server mit Plan-Ablaufdatum (ohne aktives Abo) |
| **Nächste Abbuchung verschieben** | Verschiebt deine nächste Abo-Zahlung um eine Anzahl Tage                | Server mit aktivem Abo (Stripe oder Paddle)    |
| **Kontoguthaben**                 | Ein Guthaben wird deinem SCNX-Konto für zukünftige Käufe gutgeschrieben | Nur für EU-Kunden                              |
| **Kostenlose Plan-Testphase**     | Teste einen höherwertigen Plan (Unlimited oder Professional) kostenlos  | Nur für Server im Starter-Plan                 |
| **KI-Coins**                      | Kostenlose KI-Coins für die KI-Funktionen deines Servers                | Alle Server                                    |
| **Mitgliedschaft**                | Eine kostenlose Level-2- oder Level-3-Mitgliedschaft                    | Alle Nutzer                                    |

## Wie löse ich meinen Gutschein ein?

1. Klicke auf den Link in deiner Care-E-Mail oder auf das Banner auf deinem SCNX-Dashboard
2. Überprüfe die Vorfallsdetails und unsere Entschuldigung
3. Wähle deine bevorzugte Entschädigung aus den verfügbaren Optionen
4. Falls kein Server vorab ausgewählt wurde, wähle, welcher Server die Entschädigung erhalten soll
5. Bestätige deine Auswahl

Deine Entschädigung wird sofort angewendet und du erhältst eine Bestätigungs-E-Mail.

## Häufig gestellte Fragen

### Warum habe ich einen Care-Gutschein erhalten?

Du hast einen Gutschein erhalten, weil unser Team festgestellt hat, dass du von einem Serviceproblem betroffen warst. Die E-Mail, die deinen Gutschein begleitet, erklärt den spezifischen Vorfall.

### Wie lange ist mein Gutschein gültig?

Care-Gutscheine sind ab dem Ausstellungsdatum **14 Tage** lang gültig. Nach diesem Zeitraum verfallen sie automatisch.

### Kann ich mehr als eine Entschädigung wählen?

Nein, jeder Gutschein erlaubt dir die Auswahl von **einer** Entschädigung. Wenn mehrere Server betroffen waren, erhältst du möglicherweise mehrere Gutscheine.

### Ich habe ein Abonnement – wie funktioniert eine Plan-Verlängerung?

Bei Abonnement-Nutzern verschiebt eine Plan-Verlängerung dein nächstes Abrechnungsdatum. Du wirst erst zum verlängerten Datum belastet, und dein Abonnement bleibt während dieses Zeitraums vollständig aktiv.

### Kann ich meinen Gutschein auf jemand anderen übertragen?

Nein, Care-Gutscheine sind an dein Konto gebunden und können nicht übertragen werden.

### Ich habe keine E-Mail erhalten – was soll ich tun?

Überprüfe deinen Spam-Ordner auf E-Mails von `care@scnx.xyz`. Du kannst auch in deinem SCNX-Dashboard nach einem Banner-Hinweis schauen. Wenn du glaubst, dass du einen Gutschein hätten erhalten sollen, kontaktiere bitte unser Support-Team unter [scnx.app/help](https://scnx.app/help).

### Was passiert, nachdem ich meinen Gutschein eingelöst habe?

Du erhältst eine Bestätigungs-E-Mail und die Entschädigung wird sofort angewendet. Wir könnten dich auch bitten, deine Erfahrung mit einer kurzen 1–5-Sterne-Bewertung zu bewerten, um uns zu helfen, uns zu verbessern.

### Ich habe weitere Fragen zu einem bestimmten Vorfall.

Bitte kontaktiere unser Support-Team unter [scnx.app/help](https://scnx.app/help) und gib die Ticket-ID an, die in deiner Care-E-Mail erwähnt wird.
